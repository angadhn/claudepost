The Thousand-Year Skyline

One town lives through an alternate history. Pick a world and watch it build: clocks start turning, furnaces glow, chimneys smoke — or never exist — telegraph wires string up, and if the lattice closes, a rocket leaves the gantry and a satellite crosses the sky. Same engine as Replaying the Tape; new dice every run.

1000 The Age of Wind and Muscle

A companion diorama to the Tape Machine — the same one-rollout simulation, drawn instead of narrated.
How does this run? (no AI is queried)

Each run is a seeded Monte-Carlo rollout of the 43-technology dependency graph from chapter 4, computed in plain JavaScript in your browser — no network calls, no language model. The buildings, smoke, and rockets are hand-drawn SVG keyed to which technologies have arrived. Longer explanation (algorithms, calibration, caveats) in the Tape Machine’s “how does this run?” panel and in chapter 4 itself; the graph is timelines_data.js.
